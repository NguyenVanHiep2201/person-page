Hướng dẫn sử dụng Git cơ bản cho người mới

Git là hệ thống quản lý mã nguồn phân tán phổ biến nhất hiện nay. Trong bài viết này, mình sẽ hướng dẫn cách sử dụng Git cơ bản gồm:

  • Khởi tạo kho Git
  • Commit thay đổi
  • Push code lên GitHub

Đầu tiên, bạn cần cài Git tại: https://git-scm.com/

Sau đó, mở terminal và chạy lệnh sau để khởi tạo dự án Git:

git init
git add .
git commit -m "Initial commit"
